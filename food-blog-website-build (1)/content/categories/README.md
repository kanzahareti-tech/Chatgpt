# Categories Page (Filters)

Location: content/categories/index.html

- This page provides a dedicated, filterable category view for recipes.
- It derives categories and cuisines from the existing dataset (js/recipes.js).
- It supports: Category, Cuisine, Dietary, Cooking Time, Difficulty filters, plus an Apply/Clear flow.
- Images are placeholders. You will replace with your own assets.
- Access this file directly via: /food-blog/content/categories/index.html (adjust based on hosting).
