Frontend skeleton for HotEasyRecipes (Next.js 14 App Router)

- Routes implemented: /, /recipes, /recipes/[slug], /search, /favorites
- GraphQL client scaffold for connecting to WordPress GraphQL at NEXT_PUBLIC_GRAPHQL_ENDPOINT
- Run:
  - npm install (in the frontend folder)
  - npm run dev

Notes:
- This is a starter scaffold. It does not include full CMS integration yet.
